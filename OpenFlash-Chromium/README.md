# OpenFlash ⚡

Premium, cross-browser flashcard extension for extracting and reviewing decks offline.

## ✨ Features
- **Smart Extraction**: Rip decks directly from **Quizlet** and **Knowt**.
- **Offline First**: Study anywhere with IndexedDB local storage.
- **Premium UI**: Vibrant dark/light modes with Sort Mode and classic review.
- **Cross-Browser**: Built for Chrome and Firefox.

## 🚀 Installation

### Chrome
1. Go to `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this directory.

### Firefox
1. Go to `about:debugging`.
2. Click **This Firefox**.
3. Click **Load Temporary Add-on...**.
4. Select `manifest.json` (or `manifest_firefox.json` if service workers are disabled).
